git：版本控制软件
分布式版本控制软件

命令：
git add:将本地文件增加到暂存区

git commit:将暂存区的内容 提交到 本地仓库  （版本库：可以叫分支（分之一、分支二、分支三、默认主分支：master））

git push:将本地仓库的内容 推送到 远程仓库

git pull:将远程仓库的内容 获取到 本地仓库
